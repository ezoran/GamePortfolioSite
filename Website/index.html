Climbing Mechanic

A climbing system for the game I've been working on. The system works by sending out a raycast in the forward direction, looking for objects that are set on a "Climbable" layer.

Command-Based Movement

The movement mechanic I created for Project Space. The player moves the cursor to select a desired unit, and then drags the cursor to the target position. Each unit has a different distance allowed.

Base Battle

An example of the base battle mechanic in Project Space. When each player is vying for the same base, the game goes from turn-based strategy to a 2D-sidescrolling battle game.
